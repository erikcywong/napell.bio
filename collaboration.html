Developer Platform

Open API for the Coffee Ecosystem

RESTful APIs, real-time MQTT streams, and webhook alerts. Built for developers, researchers, and supply chain partners.

GET /v1/modules/{id}/telemetry

Real-time sensor data for a specific farm module. Returns JSON with temp, EC, pH, humidity, PAR, and fog cycle status. Rate limit: 100 req/min.

GET /v1/yield/forecast

7-day, 30-day, and 180-day yield predictions per module or farm-wide. Includes confidence interval and weather-independent variance.

POST /v1/webhooks/subscribe

Subscribe to real-time alerts: threshold breaches, harvest readiness, equipment faults. Webhook POST to your endpoint within 500ms of trigger.

GET /v1/batch/{id}/trace

Full traceability data for a cherry batch: genetics source, sensor logs, nutrient formulation, harvest timestamp, and QA cupping score.

SDKs: Python Node.js Go curl
